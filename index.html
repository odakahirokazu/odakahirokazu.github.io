Software Projects

ANL Next

ANL Next is a framework to generate software which performs event-by-event data analysis.

Compton Soft

An integrated simulation and analysis software package for semiconductor radiation detectors including Compton cameras. This software provides (1) processing data obtained with multi-layer semiconductor detector systems, (2) Monte Carlo simulations of the semiconductor detector systems, and (3) data reduction/analysis including Compton reconstruction.

MONACO

Monte-Carlo simulations for astrophysics and cosmology

RubyROOT

A Ruby binding of ROOT, a widely used data analysis framework developed by CERN for high-energy physics.
